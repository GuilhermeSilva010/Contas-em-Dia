Documentação Software Contas em Dia

Documentação Técnica.

o Sistema permite criar um novo usuario e criar um login no sistema, para poder começar a registrar suas contas. Após criar seus dados e se logar no sistema você tem um CRUD para inserir dados das divídas.

Interface de Front-end

Camada responsável por coletar as informações do usuário e dados da conta a ser adicionada no sistema. Nessa camada que está sendo feita a validações de senha entre outras, para melhorar a usabilidade do usuário. (Todas classes no código com nomes iniciais “tela” estão relacionadas ao FrontEnd.)

BackEnd For Front-end

Camada responsável por receber as informações da camada anterior e desacoplar o código, nessa camada está sendo utilizada apenas para repasse das informações e seguir os conceitos arquitetural de boas práticas. (Classe responsável por esse serviço está nomeada de BFFUsuario)

BackEnd

Camada responsável por receber as informações da camada anterior e desacoplar o código, nessa camada está sendo utilizada apenas para repasse das informações e seguir os conceitos arquitetural de boas práticas. Como não executamos nenhuma regra de negócio no sistema essa camada ficou com responsabilidade apenas de repassar as informações para a camada de dados. (Classe responsável por esse serviço está nomeada de BEUsuario)

Dados

Responsável por receber as informações da camada anterior e fazer conexão e interações com a base de dados. (Classe responsável por essa função está nomeada de Dados)
